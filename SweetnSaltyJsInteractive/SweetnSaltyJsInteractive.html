Welcome To Interactive Sweetn'Salty Game


About
sweet’nSalty is a game where the coumputer will print numbers from given starting number to end numbers. When it hits certin number it
will either prints "Sweet", "Salty", or "sweet’nSalty". After it is done printing, then it will list the number of "Sweet", "Salty", and "sweet’nSalty" has been printed.


Here are the basic rules for Sweetn'Salty Game
  1. Enter a starting number, must be positive number.
  2. Enter an end number, must also be positive number, also 200 more than & 10,000 less than prevoius Number.
  3. If the first number is 1 then second number must be atleast 200 and should not be more than 10,000
  4. Enter a third positive number to specify how many number per line you want to print.
  5. The third number must be greater than 1/100 of the total number
  6. For example if first Number is 1, and sencond number is 1000 than third number must be greater than 10
